# HITL UI

React/Vite frontend for nurse review of extracted criteria and evidence.

## Responsibilities

- Display protocols, criteria, and evidence snippets.
- Provide SNOMED candidate review and edit workflows.
- Capture nurse feedback for audit and retraining.

## Development

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start the Vite dev server.
- `npm run build` - Production build.
- `npm run test` - Run unit tests (where available).

## Structure

- `src/screens` - Top-level routes/views.
- `src/features` - Feature-specific UI modules (protocols, mapping).
- `src/components` - Shared UI components.
- `src/design-system` - Tokens, theme, and UI primitives.

## Notes

The UI is wired to the API endpoints documented in `docs/api/api_spec.md`.
